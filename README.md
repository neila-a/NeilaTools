## Introduction - 介绍

### Summary - 概要

Neila的一些没用工具的平台。

### Features - 特性

- 全部采用响应式布局。
- 使用PWA以在桌面放置，快捷打开。
- 使用CacheStorage达到超快的访问速度。

## Q&A

- Q：为什么要给Cylinder单独设计一套渲染系统？  
  A：因为Cylinder是方格驱动的，而市面上的渲染系统都是能支持各种图形的，显然带来了额外的性能开销。
