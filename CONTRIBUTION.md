# Contribution

## Create a new tool

- Write a TSX file in `/tools/`;
- Add to `/components/tools/info.tsx`; 
- Add to `/components/tools/component.tsx`. 

## Add a new language

- Write a JSON file in `/locales/`;
- Add to `/pages/_app.tsx`.

## Make a commit

- Do something;
- `npm run modifyPackage`
- Push it to GitHub.
